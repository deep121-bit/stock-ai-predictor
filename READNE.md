# 📊 AI Stock Price Prediction System

## 🚀 Overview
This is an AI-powered Stock & Crypto Price Prediction web app using LSTM (Deep Learning) and Flask.

It predicts next-day stock prices based on historical market data.

---

## 🧠 Features
- Real-time stock data fetching (Yahoo Finance)
- LSTM deep learning model
- Feature engineering (MA, volatility, returns)
- Web UI using Flask
- Actual vs Predicted graph visualization
- Supports stocks & crypto (AAPL, TSLA, BTC-USD)

---

## 🛠️ Tech Stack
- Python 🐍
- TensorFlow / Keras
- Flask
- Pandas / NumPy
- Matplotlib
- yFinance
